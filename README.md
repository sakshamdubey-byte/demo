# Rock vs Mine Prediction using Sonar Data

This repository contains a machine learning project that classifies sonar return signals to determine whether the detected object is a **Rock** or a **Metal Cylinder (Mine)**.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/sakshamdubey-byte/demo/blob/main/Rock_vs_Mine_Prediction.ipynb)

## Project Overview
* **Model:** Logistic Regression
* **Dataset:** Sonar Dataset (60 features of sonar frequencies)
* **Training Accuracy:** ~83.4%
* **Test Accuracy:** ~76.2%
